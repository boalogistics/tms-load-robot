Multiple Load Import
  • 1. Please note each row in the template may be marked as L (Load) or C (Commodity) in the column "LoadType".
  • 2. To create a load section fill out the desired data marking the row as "L"; this section will be held to the requirements of row type L.
  • 3. Each subsequent row marked with a "C" will be treated as a commodity line and added to the above load grouping; this section will be held to the requirements of row type C.
  • 4. When using the template it is required that intended loads and their commodities be grouped together.

Required Columns Row Type L: LoadType, ClientID, sid, PickupDate, OrigZip, DescZip, Pieces, Packaging, Weight, CarrierSCAC
Commodity Pricing (Optional): retail, retaildisc, cost, costdisc
Fuel Accessorial (Optional): FuelSurchargeRetail, FuelSurchargeCost
Shipment (Optional): POReference, ShipperNo, ProNumber, PickupDockCloseTime, DeliveryDate, Palletized, Stackable,Pallets, PalletSize, Linearfeet,Shipmentvalue, Comments, CarrierName, TrailerTypeID, PayerDescription, Notes
Origin Address (Optional): OrigCompany,OrigAddress1,OrigAddress2,OrigCity,OrigState,OrigCountry,OrigContact,OrigEmail,OrigPhone,OrigFax,OrigNotes,OrigLocationType
Destination Address (Optional): DescCompany,DescAddress1,DescAddress2,DescCity,DescState,DescCountry,DescContact,DescEmail,DescPhone,DescFax,DescNotes,DescLocationType

Required Columns Row Type C: Pieces,Packaging,Class,Weight
Optional: Description, NMFC, Density, Length, Width, Height, UnitsWeight, UnitsDim, UnitsDensity

Unit Dimension Values:
  • 0 : Feet
  • 1 : Inches
  • 2 : Meters
  • 3 : Yards
  • 4 : CM
  • 5 : PCF
  • 6 : kgm
  • 10: lbs
  • 11: kg
Shipment Mode Values:
  • 1 : LTL
  • 2 : Truckload
  • 3 : Volume
  • 4 : Intermodal
  • 5 : Ocean
  • 6 : Drayage
  • 10: Air
  • 11: RF LTL
Pallet Size Values:
  • 0 : 48 x 48
  • 1 : 48 x 42
  • 2 : 48 x 40
  • 3 : 44 x 42
  • 4 : 42 x 42
  • 5 : Non Std
  • 6 : 40 x 24

*TrailerTypeIDs can be found under Setup->Equipment Types*
*Package types are the values enabled in your current system
*PalletSize (Optional): If value is blank, the shipment will default to "0" (48x48)
*Date format is MM/DD/YYYY
*The uploader currently only supports two stops, origin and destination.
